# Análisis del Mercado Laboral para Data Analyst

Proyecto end-to-end de análisis del mercado laboral para el rol de Data Analyst,
usando datos reales obtenidos via API y scraping.

## Stack
- Python
- SQL / SQLite
- Power BI

## Fuentes de datos
- Adzuna API (Estados Unidos, Reino Unido, Australia, Canadá, Brasil, México)
- Computrabajo scraping (Argentina)
